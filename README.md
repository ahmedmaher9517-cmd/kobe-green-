# KOBE GREEN ERP - Supabase Cloud Ready

Multi-tenant coffee trading ERP system with cloud database support.

## Features
- ☕ Coffee inventory management (green beans, roasted, ground)
- 📦 Sales & Purchase management
- 💰 Treasury & cash flow
- 📊 Reports & Analytics  
- 🌐 E-commerce integration (Amazon FBA, Noon FBN)
- ☁️ Cloud database (Supabase PostgreSQL)

## Quick Start

### Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
```

### Deploy to Streamlit Cloud
1. Fork/upload this repo
2. Deploy on share.streamlit.io
3. Add Secrets:
```toml
SUPABASE_DB_URL = "postgresql://..."
```

## Tech Stack
- Streamlit
- Supabase (PostgreSQL)
- SQLite (local dev)
- Plotly (charts)
